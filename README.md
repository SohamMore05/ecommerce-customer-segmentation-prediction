# E-Commerce Customer Segmentation & Prediction

## Project Overview

This project focuses on customer segmentation and prediction using machine learning techniques on e-commerce transaction data. The objective is to identify meaningful customer groups based on purchasing behavior and support targeted marketing strategies.

## Key Features

* Data Cleaning and Preprocessing
* Exploratory Data Analysis (EDA)
* RFM (Recency, Frequency, Monetary) Analysis
* K-Means Clustering
* Hierarchical Clustering
* DBSCAN Clustering
* Machine Learning Models
* Power BI Dashboard
* Customer Segment Prediction Interface

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn
* Power BI

## Customer Segments

* VIP Customers
* Loyal Customers
* Potential Customers
* At-Risk Customers

## Machine Learning Models

* Logistic Regression
* Decision Tree
* Random Forest
* K-Nearest Neighbors (KNN)
* Gradient Boosting

## Results

* Successfully segmented customers into four meaningful groups.
* Achieved approximately 97% prediction accuracy.
* Developed an interactive Power BI dashboard for business insights.
* Built a customer prediction interface using RFM features.
